Facial Emotion Tracking System using AI

Developed a real-time AI-powered application that detects and interprets facial emotions via webcam input. Leveraged MediaPipe for accurate facial landmark detection, OpenCV for real-time video processing, and pyttsx3 for audio-based emotion feedback. The system computes mood scores based on facial expressions and visualizes the user's emotional trends using Matplotlib.

Built a live feedback loop to improve emotional awareness and self-monitoring.
Enhanced emotional recognition accuracy with tailored facial action unit analysis.
Created an intuitive daily mood tracking chart for visualization of emotion patterns.

Tech Stack: Python, OpenCV, MediaPipe, pyttsx3, Matplotlib
Core Skills: Real-Time AI Inference, Emotion Scoring, Visualization, Voice Feedback

ResumeBuddy – LLM-Based Resume Builder Chatbot

Created an intelligent chatbot tailored for job seekers to generate ATS-optimized resumes. Utilized OpenAI's GPT models and customized prompt engineering techniques (Chain-of-Thought, Zero-Shot, Few-Shot) to extract keywords from job descriptions and tailor resumes accordingly. Integrated logic to identify required skills, map them to user profiles, and generate human-like, recruiter-ready resumes.

Achieved 95% accuracy in matching resumes to job roles.
Integrated job-matching logic and contextual skill suggestions.
Ensured output quality aligned with modern ATS and recruiter expectations.

Tech Stack: OpenAI GPT, Python
Core Skills: LLM Prompt Design, NLP, Resume Optimization, Career Support Automation
